# Personal Milestones

Foundry VTT v13 module scaffold for per-Actor checklist milestones.

## Current Status
- Repo standards file created in copilot-instructions.md
- TypeScript + esbuild scaffold created
- Module manifest added for Foundry v13
- Actor sheet button opens a checklist popup app
- Checklist data persists on Actor flags
- Add, remove, and complete milestone actions implemented

## Setup
1. Install dependencies:
   npm install
2. Build module bundle:
   npm run build
3. Rebuild while developing:
   npm run watch
4. Type-check source:
   npm run typecheck
